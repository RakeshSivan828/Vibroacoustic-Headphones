# 🛠 Troubleshooting Guide

---

# No Vibration

Possible Causes

- Battery discharged
- Wiring issue
- ESP32 not powered
- Incorrect PWM configuration

---

# Wrong Frequency Detection

Possible Causes

- Weak audio signal
- Incorrect sampling rate
- Speaker wiring issue

---

# Continuous Vibration

Possible Causes

- Incorrect threshold
- Incorrect frequency mapping

---

# ESP32 Not Detected

Possible Causes

- USB Driver
- Faulty USB cable
- Wrong COM port

---

# Motors Do Not Stop

Possible Causes

- PWM value not reset
- Frequency outside expected range

---

# Power Issues

Check:

- Battery voltage
- XL6009 output
- Common ground
